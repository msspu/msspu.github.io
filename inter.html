Interaktivní graf

v R je integrována celá řada nástrojů z JS: viz https://www.htmlwidgets.org/

Zde používám Dygraph, s posuvníkem roků na časové řadě, používám stejná data jako ve vývoji zaměstnanecké struktury 1995 až 2018. A ukazuji jak od roku 1995 narůstal podíl OSVČ bez zaměstnanců, ale podíl podnikatelů se zaměstnanci stagnoval, nebo mírně klesal.

library(readxl)
library(tidyr)
library(dplyr)
library(dygraphs)
library(xts)
dataPost <- read_excel("h:/moje/DATA/Post95to18.xlsx")
dataPost$year <- as.Date(paste(dataPost$REFTROK, 1, 1, sep = "-"))

Před použitím dygraph je potřeba data převést na datový typ xts (time series)

Formátování grafu je poměrně strohé, např. současné nastavení umožňuje na ose x nastavení roční - ANNUAL nebo desetiletí DECADAL. Zde by se hodila spíš pětiletka, ale to by bylo potřeba nastavit pomocí JS zde: https://github.com/danvk/dygraphs/blob/master/src/dygraph-tickers.js#L221 proměnná TICK_PLACEMENT

dg1 <- dplyr::select(dataPost, Podnikatel, OSVC)

dg1 <- as.xts(ts(start = c(1995), end=c(2018),
  data =  dg1))
dygraph(dg1)  %>% 
dyRangeSelector(dateWindow = c("1995-01-01", "2018-01-01")) %>% 
dyLegend(width=500, show="auto")
